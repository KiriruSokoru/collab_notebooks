Security_scaner.ipynb
This file helps quickly manage access to spreadsheets in a specific folder. You need to have a spreadsheet with valid, up-to-date email addresses. 
Based on this email spreadsheet, the script will scan through the folder, revoke access if it doesn't find a required email, and grant editor role access if the email is present in the file. 
Important: for the script to work correctly, you'll need to authenticate and create a key in Google Cloud.

json_extractor.ipynb
This file helps process large amounts of JSON format data (Telegram channel dumps). 
In this specific variation, the script was used to classify questions and problems encountered by users. 
The script helps identify weak points, create guides, and highlight issues related to specific users.
